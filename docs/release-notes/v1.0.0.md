# SecureCheck AI Web Scanner v1.0.0

This is the first public open-source release of SecureCheck AI Web Scanner.

SecureCheck is a free Chrome Extension Manifest V3 tool for scanning AI-built and vibe-coded websites for browser-visible security risks. It turns findings into copy-ready prompts that can be pasted into Codex or another AI coding assistant.

## Highlights

- Passive live-site scan from the active browser tab.
- HTTP security header checks.
- Exposed file and endpoint checks.
- Cookie security attribute checks.
- DOM and same-origin JavaScript bundle checks.
- Frontend secret pattern masking.
- One-click AI remediation prompt per finding.
- Copy-all report for AI-assisted repair workflows.
- English and Traditional Chinese UI.
- Local-first privacy model with no analytics, ads, telemetry, backend, or paywall.

## Chrome Extension Package

The GitHub release includes a generated extension zip asset:

```text
securecheck-ai-web-scanner-v1.0.0.zip
```

For local development, Chrome can also load the unpacked repository folder directly from `chrome://extensions`.

## Known Limitations

SecureCheck only detects issues visible from the browser. It is not a replacement for a full security audit, backend code review, dependency audit, or penetration test.
