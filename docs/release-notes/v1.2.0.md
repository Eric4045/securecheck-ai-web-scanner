# SecureCheck AI Web Scanner v1.2.0

This release prepares SecureCheck AI Web Scanner for professional public review, Codex for Open Source evaluation, and Chrome Web Store resubmission.

SecureCheck AI Web Scanner is a free Chrome Extension Manifest V3 tool for scanning AI-built websites for browser-visible security risks. It turns findings into copy-ready prompts that can be pasted into Codex or another AI coding assistant.

## Highlights

- Standardized public naming around `SecureCheck AI Web Scanner`.
- Updated Chrome extension display name and description.
- Replaced casual UI copy with more professional AI-built website security language.
- Updated demo screenshots and example report with fictional `.example` data.
- Kept the local-first privacy model: no analytics, ads, telemetry, backend, or paywall.
- Preserved passive scan behavior and temporary site-permission handling.

## Chrome Extension Package

The GitHub release includes a generated extension zip asset:

```text
securecheck-ai-web-scanner-v1.2.0.zip
```

Use this package for Chrome Web Store resubmission unless the Developer Dashboard already requires a higher version.

## Known Limitations

SecureCheck only detects issues visible from the browser. It is not a replacement for a full security audit, backend code review, dependency audit, or penetration test.
